# Libstackz
Library Management System

# Features
- Create and Manage Multiple Libraries
- Manage Books and Catalogues
- Import Data with Excel.
- Addons and Feature Enhancements.
- Email Functions
- Member Management
- Metadata Management

## Access URL
[app.libstackz.com](https://app.libstackz.com)

## Tech Stack
- Next.js
- Tailwind
- Postgresql
- Prisma ORM
- Redis
- Manitine UI
- Ziqx Auth
- Ziqx Drive
