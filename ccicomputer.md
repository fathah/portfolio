# CCI Computer Global
Global Refurbished Computer Provider with Wharehouses in Canada, Hongkong and the UAE. 

## Access Link
[ccicomputerglobal.com](https://ccicomputerglobal.com)

## Features
- Landing Page
- Client Dashboard
- Admin Dashboard
- Sales Dashboard
- Access Lot and Invenories for Sales
- Sales Contact Panel
- Admin Monitoring
- Business Whatsapp Panel
- Customer Management System
- Inventory Management System

## Tech Stack
- Next.js
- MySQL
- Ubuntu VPS
- Tailwind
- Prisma
